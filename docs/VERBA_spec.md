# THE CURTAIN - Dating App for Introverts

## Tagline
*"What you say matters, not how you look"*

---

## Core Features (UPDATED)

### 1. The Curtain - Photos by Permission
- No photos on profile by default
- Photos visible only when BOTH people agree
- Mutual consent to reveal photos
- Until then - only name, age, bio, voice note (optional)
- You can see their interest (mutual like) but photos stay hidden

### 2. Ghost Mode - True Privacy
- No online status
- No "last seen"
- No read receipts
- Completely anonymous
- No one knows you're there

### 3. Slow Lane - Quality First
- Only 1 active chat at a time
- Can't browse while chatting
- Focus on one person
- Can't spread yourself thin

### 4. The Writer - Text-First (Voice Optional)
- Bio in form of story/poem/thought
- Match by writing style
- Voice note intro - COMPLETELY OPTIONAL
- Written conversations over quick texts

### 5. No Pressure
- No photos required on profile
- No pressure to reveal photos
- No pressure for video calls
- Take your time

### 6. Video Call - Mutual Consent
- Video call only when BOTH agree
- Either person can request
- Other person must accept
- No pressure to go on camera

---

## User Flow

| Step | What Happens |
|------|--------------|
| 1 | Sign up - Quiz (personality, interests) |
| 2 | Profile - Write a story/bio (voice note optional) |
| 3 | Browse - See profiles WITHOUT photos |
| 4 | Like/Connect - Send intro message |
| 5 | If accepted - chat opens |
| 6 | Photos revealed - ONLY when BOTH agree |
| 7 | Video call - ONLY when BOTH agree |
