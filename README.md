# VERBA

*A dating app for introverts*

### Tagline
**"Words before looks"**

---

## The Concept

Dating app where photos stay hidden until BOTH people agree to reveal them.

---

## Core Features

| Feature | Description |
|---------|-------------|
| **The Curtain** | Photos hidden until mutual consent |
| **Ghost Mode** | No online status, no last seen |
| **Slow Lane** | 1 chat at a time |
| **The Writer** | Bio as story/poem (voice optional) |
| **Video** | Only when BOTH agree |
